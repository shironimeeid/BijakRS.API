RS Bed Covid API

  • Dapatkan semua data provinsi:   https://rs-bed-covid-api.vercel.app/api/get-provinces
  • Dapatkan data kota atau kabupaten yang ada di provinsi tersebut:   https://rs-bed-covid-api.vercel.app/api/get-cities?provinceid=51prop
  • Dapatkan data rumah sakit berdasarkan provinsi dan kota atau kecamatan:   https://rs-bed-covid-api.vercel.app/api/get-hospitals?provinceid=51prop&cityid=5171&type=1
  • Dapatkan data detail kapasitas ruangan atau kasur berdasarkan rumah sakit:   https://rs-bed-covid-api.vercel.app/api/get-bed-detail?hospitalid=5171133&type=1
  • Dapatkan data peta rumah sakit:   https://rs-bed-covid-api.vercel.app/api/get-hospital-map?hospitalid=5171133
NB: Untuk melihat daftar id provinsi, Anda bisa menggunakan API get provinces terlebih dahulu.
NB: Untuk melihat daftar id kota atau kecamatan, Anda bisa menggunakan API get cities terlebih dahulu.
NB: Query string type memiliki 2 optional value: 1 untuk covid dan 2 untuk non-covid
NB: Untuk mendapatkan hospital id, Anda bisa menggunakan API get hospitals terlebih dahulu
Repositori github: https://github.com/satyawikananda/rs-bed-covid-indo-api/
